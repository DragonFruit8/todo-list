Week 1

# Installation instructions
1) Download repository
2) Extract and save folder on local computer
3) Use command prompt OR git bash(linux-based) command prompt to:
   + Navigate to local directory (C:/path/to/folder/)
   - Type: `npm install`
   - Type: `npm run dev`
   + npm run dev
